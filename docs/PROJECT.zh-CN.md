# Redox RFB Predictor：项目说明

## 一句话

输入一个有机分子的 SMILES，预测它作为**水系有机液流电池**候选物的标准氧化还原电位（V vs SHE）。

```text
SMILES
  ↓
分子特征 / 量子化学特征
  ↓
机器学习模型
  ↓
预测的氧化还原电位（V vs SHE）
```

## 它解决什么问题

液流电池需要寻找合适的氧化还原活性分子。候选分子空间很大，逐个做 DFT 或实验都很慢。这个项目提供两个速度与物理信息量不同的预测器：

| 模型 | 输入 | 适合做什么 |
| --- | --- | --- |
| `rdkit` | 26 个 RDKit 描述符 + 1024 位 Morgan 指纹 | 快速筛选大量分子，约毫秒级/分子 |
| `hybrid` | 26 个 RDKit 描述符 + 7 个 GFN2-xTB 量子描述符 | 对少量候选分子做带电子结构信息的复筛，约 10–60 秒/分子 |

示例：对对苯醌 `O=C1C=CC(=O)C=C1`，快速模型会返回一个预测的电位数值；混合模型会先运行 xTB，再给出另一个预测值。

## 数据和标签

训练数据来自 **RedDB**：面向水系液流电池的 DFT 计算分子数据库。项目使用其中 15,673 个清洗后的有机分子，主要是醌类和含氮芳香体系。

标签不是实验电位，而是根据 RedDB 的 DFT 反应能换算的标准电位：

```text
E° (V vs SHE) = − reaction energy (Hartree) × 27.2114 / 2
```

因此，这个模型的数值应理解为：**对 RedDB 的 DFT 电位尺度的预测**。它可用于同一化学域内的排序和筛选；不能直接当作实验半电池电位，也不应直接外推到 TEMPO、viologen、金属配合物等不在训练域内的化学体系。

## 已随项目交付的结果

在 129 个可成功计算 xTB 特征的分子上，重复交叉验证得到：

| 特征 | MAE (V) | R² |
| --- | ---: | ---: |
| xTB only | 0.203 | 0.29 |
| RDKit descriptors | 0.204 | 0.37 |
| RDKit descriptors + fingerprint | 0.194 | 0.43 |
| RDKit + xTB | 0.173 | 0.50 |

这支持一个有限但有用的结论：在这 129 个分子的共同评估集上，xTB 的电子结构特征提高了预测表现。它**不**证明该模型已在真实实验体系或所有电池分子上有效。

## 与 SciencesLoop Lab 的关系

这是一个独立、可公开的工具型项目；`sciencesloop-lab` 则记录长期研究问题、实验计划、文章和跨项目比较。

下一步研究问题不是立刻堆更多模型，而是：

> 在按分子骨架划分、或在真正未见的电池分子家族上，RDKit、xTB 混合模型和预训练分子模型谁还能保持优势？
